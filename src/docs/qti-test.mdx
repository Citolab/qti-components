### Load test

Use a `<test-container></test-container>` to load and encapsulate the `qti-assessment-test` webcomponents.

Use the following file structure
A qti-test loads a QTI3.0 assessmenttest.xml file from a package folder.

```plaintext
Root/
├── index.html
└── /assets/api/examples/
    ├── assessmenttest.xml
    └── imsmanifest.xml

```
