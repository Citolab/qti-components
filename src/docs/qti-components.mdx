## Quick Start

Add the following code to your page.

```html
<link
  rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/@shoelace-style/shoelace@%VERSION%/%CDNDIR%/themes/light.css"
/>
<script
  type="module"
  src="https://cdn.jsdelivr.net/npm/@shoelace-style/shoelace@%VERSION%/%CDNDIR%/shoelace-autoloader.js"
></script>
```

Now you have access to all of Shoelace's components! Try adding a button:

```html:preview:expanded:no-codepen
<sl-button>Click me</sl-button>
```
