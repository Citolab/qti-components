# ðŸ‘‹ HI QTI

A set of test interactions to be used in the educational field.

- use the interactions in standard form elements
- use qti-item to render test questions with advanced scoring
- use qti-test to render a complete qti-test including navigation

<Canvas of={Stories.Default} />

## Installation

```sh
npm install @citolab/qti-components
```

Import the webcomponents

```js
import '@citolab/qti-components/qti-components';
import '@citolab/qti-components/qti-test/core';
import '@citolab/qti-components/qti-test/components/test-next';
import '@citolab/qti-components/qti-test/components/test-prev';
```
