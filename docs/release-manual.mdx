# Release Manual

## 1) Release changed `@qti-components/*` packages

```bash
pnpm run release:changed:dry
pnpm run release:changed
pnpm run publish:missing
```

This releases and publishes all changed non-umbrella packages together.

## 2) Release umbrella `@citolab/qti-components`

Run GitHub Actions workflow `Manual release` and choose:
- `branch`
- `dry_run`
- `version_mode`
- optional `exact_version`

This workflow is umbrella-only and publishes umbrella when a new umbrella tag is created.

## 3) Fallback publish command

```bash
pnpm run publish:missing:all
```

Use only when you explicitly want missing publishes including umbrella.
